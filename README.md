# Offline Activation — Telegram Mini App

`index.html` — весь магазин в одном файле. Ничего устанавливать не нужно, сборка не требуется.

---

## Что понадобится перед стартом

- Аккаунт на **GitHub** (бесплатно, github.com)
- Бот в Telegram, созданный через **@BotFather** (если ещё нет — напиши ему `/newbot` и следуй инструкциям, в конце он даст токен, он тебе тут не понадобится, только сам факт что бот создан)

---

## Способ A — без терминала, через сайт GitHub (проще всего)

1. Зайди на **github.com**, войди в аккаунт.
2. Нажми **New** (зелёная кнопка сверху слева, или "+" в правом верхнем углу → "New repository").
3. Введи любое имя репозитория, например `offline-activation`. Repository должен быть **Public** (для бесплатного GitHub Pages). Остальные настройки не трогай → **Create repository**.
4. На странице пустого репозитория нажми **uploading an existing file** (ссылка прямо на странице) — либо кнопку **Add file → Upload files**.
5. Перетащи туда файл **`index.html`** (важно: имя файла должно остаться именно `index.html`, не переименовывай).
6. Внизу нажми **Commit changes** (можно просто оставить всё по умолчанию).
7. Теперь включаем сайт: сверху репозитория зайди в **Settings** → слева в меню найди **Pages**.
8. В разделе **Build and deployment → Source** выбери **Deploy from a branch**.
9. Ниже, в **Branch**, выбери `main` и папку `/ (root)` → нажми **Save**.
10. Подожди 30–90 секунд, обнови страницу — сверху появится зелёный блок с надписью **"Your site is live at..."** и ссылкой вида:

    ```
    https://твой-юзернейм.github.io/offline-activation/
    ```

11. Открой эту ссылку в обычном браузере (на телефоне или компьютере) — должен открыться сам магазин. Если открылось — всё сделано верно, идём дальше к настройке бота.

## Способ B — через git в терминале (если умеешь пользоваться git)

```bash
git init offline-activation
cd offline-activation
# скопируй сюда index.html
git add index.html
git commit -m "Offline Activation mini app"
git branch -M main
git remote add origin https://github.com/твой-юзернейм/offline-activation.git
git push -u origin main
```

Дальше — те же шаги 7–11 из Способа A (включить Pages в Settings).

## Подключаем ссылку к боту в Telegram

1. Открой **@BotFather** в Telegram.
2. Напиши `/mybots` → выбери своего бота из списка.
3. Нажми **Bot Settings** → **Menu Button** → **Configure Menu Button**.
4. Пришли ту самую ссылку с GitHub Pages (`https://твой-юзернейм.github.io/offline-activation/`), затем пришли название кнопки, например `Открыть магазин`.

   *Альтернатива:* команда `/newapp` в BotFather создаёт полноценное Mini App с иконкой и описанием (не просто кнопку меню) — тоже подойдёт, разница только в способе запуска для пользователя.

5. Открой своего бота в Telegram (обычным поиском) → должна появиться кнопка меню снизу у поля ввода → нажми на неё. Приложение должно открыться прямо внутри Telegram.

## Как обновлять приложение после изменений

Просто загрузи новую версию `index.html` в тот же репозиторий тем же способом (перетащить файл → Commit changes, он предложит заменить существующий). GitHub Pages подхватит изменения автоматически за 1–2 минуты. В самом Telegram иногда нужно закрыть и заново открыть Mini App (или потянуть вниз), чтобы обновление подгрузилось — Telegram кэширует страницу.

---

## Перед реальным запуском — обязательно проверь

- В начале файла `index.html`, в объекте `CONFIG`, находится:
  ```js
  adminUsernames: ["RutuGames", "RutuLljko"],
  ```
  Это единственные два аккаунта, которым виден раздел администратора. Проверь, что юзернеймы указаны верно (без `@`, точно как в Telegram).

- `paymentProvider: "mock"` — оплата пока имитационная. Для настоящих денег (Telegram Stars или банковские карты) нужен реальный бэкенд — в начале `<script>` есть подробный комментарий, что именно менять.

## Если что-то не работает

- **Ссылка открывается, но белый экран** — обнови страницу через 1–2 минуты после Commit, GitHub Pages иногда собирается не мгновенно.
- **404 на GitHub Pages** — проверь, что файл называется ровно `index.html` и лежит в корне репозитория (не внутри папки), а в Settings → Pages выбрана ветка `main` и папка `/ (root)`.
- **В Telegram кнопка меню не открывает приложение** — убедись, что ссылка начинается с `https://` (не `http://`), Telegram не откроет Mini App без HTTPS.
- **Изменения не видны в Telegram** — закрой чат с ботом полностью и открой заново, либо подожди пару минут, Telegram кэширует Mini App.
